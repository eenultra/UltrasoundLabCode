PRECISION ACOUSTICS LTD. 

RADIATION FORCE BALANCE INSTALLATION INSTRUCTIONS

*************************************************



1. Run "RFB 2 INSTALLER.EXE" to unpack installation files to a user-specified location
2. Run "VOLUME\SETUP.EXE"
2. Follow on-screen instructions for the installation of NI software
3. Run "DEVICE PLUGINS.EXE" and extract the files to
   "\DOCUMENTS AND SETTINGS\ALL USERS\APPLICATION DATA\RFB\DEVICE PLUGINS" for WinXP and
   "\PROGRAMDATA\RFB\DEVICE PLUGINS" for Win7 
4. Run "USBDRIVERINSTALLER.EXE" if using a Mettler-Toledo balance
